# 📅 Agenda de Contatos em Python

Um sistema de agenda de contatos simples e eficiente desenvolvido em Python, rodando direto no terminal. O projeto foi criado para praticar conceitos de lógica de programação, manipulação de estruturas de dados (listas e dicionários) e operações de CRUD.

## 🚀 Funcionalidades

* 📝 **Cadastrar Contatos:** Salva nome, telefone e e-mail.
* 📋 **Visualizar Agenda:** Lista todos os contatos cadastrados.
* 🔄 **Atualizar Dados:** Permite alterar as informações de um contato existente.
* ❌ **Deletar Contatos:** Remove contatos da agenda de forma segura usando índices.
* ⭐ **Sistema de Favoritos:** É possível favoritar/desfavoritar contatos e listar apenas os favoritos com proteção de índice.

## 🛠️ Tecnologias Utilizadas

* **Python 3** (Lógica principal e estruturas de dados)

## 💻 Como Rodar o Projeto

1. Certifique-se de ter o Python instalado na sua máquina.
2. Baixe o arquivo `Agenda de contatos.py`.
3. Abra o terminal na pasta do arquivo e execute o comando:
   ```bash
   python "Agenda de contatos.py"
